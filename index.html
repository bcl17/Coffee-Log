Google Sheets Integration
Every time you save a shot, it can be sent automatically to a Google Sheet. Here's how to set it up in about 5 minutes.

Status

Not connected — enter your Web App URL below to connect

Step-by-step setup
Follow these steps once — takes about 5 minutes.

1. Go to sheets.google.com and create a new spreadsheet.
2. Name the first sheet tab: Shots
3. In the spreadsheet, go to Extensions → Apps Script
4. Delete any existing code and paste in the script below:

function doPost(e) { var sheet = SpreadsheetApp .getActiveSpreadsheet() .getSheetByName('Shots'); // Add headers if sheet is empty if (sheet.getLastRow() === 0) { sheet.appendRow([ 'Date','Bean','Roast','Grinder', 'Grind Size','Dose (g)','Yield (g)', 'Time (s)','Ratio','Taste Notes', 'Rating','Notes' ]); } var d = JSON.parse(e.postData.contents); sheet.appendRow([ d.date, d.bean, d.roast, d.grinder, d.grindSize, d.dose, d.yield_g, d.time, d.ratio, d.taste, d.rating, d.notes ]); return ContentService .createTextOutput(JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); }

5. Click Deploy → New deployment
6. Type: Web app
7. Execute as: Me · Who has access: Anyone
8. Click Deploy → copy the Web App URL and paste it above.
